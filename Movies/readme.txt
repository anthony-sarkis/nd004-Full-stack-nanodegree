
Documentation for Movie trailer website
-------------


Files:
------------------------------------------------------------------------
Data and basic functions --> entertainment_center.py
Classes --> media.py
HTML/CSS/JS bulder -> fresh_tomatoes.py


Description:
------------------------------------------------------------------------
The program generates an HTML movie trailer website. The primary functions are:
-> Display movie poster, title, and storyline.
-> Upon clicking any of the above information, the movie's trailer will play.


Getting started:
------------------------------------------------------------------------
1. Unzip and open the "Movies" folder
2. Check you have Python 3.5 installed, 
if not you can download it here: https://www.python.org/downloads/
3. Run entertinament_center.py

The movie site will attempt to automatically load in your browser,
you may also manually view or use the file by navigating to the Movies folder,
the file is named "fresh_tomatoes.html".
